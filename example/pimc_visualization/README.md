# PIMC Worldline Visualization

This directory contains tools for visualizing PIMC (Path Integral Monte Carlo) worldline configurations in 3D.

## Overview

The visualization notebook displays worldline paths from PIMC simulations, showing:
- 3D coordinate space traces of bead positions
- Color-coded cycles by cycle length
- Interactive exploration of multiple configurations
- Support for both PIMC output files and minimization trajectories

## Requirements

Install the required visualization dependencies:

```bash
pip install plotly ipywidgets
```

For Jupyter Lab, you may also need:
```bash
jupyter labextension install jupyterlab-plotly
```

## Usage

### Running the Notebook

1. Start Jupyter:
   ```bash
   jupyter notebook vis-worldlines.ipynb
   ```
   or
   ```bash
   jupyter lab vis-worldlines.ipynb
   ```

2. Run all cells to load and visualize PIMC configurations

### Input Files

The notebook can visualize:

**PIMC worldline files** (`ce-wl-*.dat`):
- Standard output from PIMC simulations
- Contains configurations at different MC steps

**Minimization trajectory files**:
- Generated by `find_local_minimum()` with `trajectory_file` parameter
- Each configuration corresponds to a minimization iteration
- Example: `tests/tmp/pimc_M3_N2_trajectory.dat`

**Test data**:
- `tests/test_data/N2-Nbeads3-cycle1.dat` - Small 2-particle, 3-bead system
- `tests/test_data/N2-Nbeads3-cycle2.dat` - Alternative cycle configuration

### Changing Input Files

Edit the `file_path` variable in the "Input" section:

```python
# Use test data
file_path = '../../tests/test_data/N2-Nbeads3-cycle1.dat'

# Or use minimization trajectory
file_path = '../../tests/tmp/pimc_M3_N2_trajectory.dat'

# Or use your own PIMC output
file_path = '/path/to/your/ce-wl-file.dat'
```

Don't forget to set the correct box dimensions:
```python
box = [10.0, 10.0, 10.0]  # [Lx, Ly, Lz] in Angstroms
```

## Features

### Interactive Visualization

Use the slider to navigate through different configurations:
- **Slider**: Select configuration index
- **3D Plot**: Rotate, zoom, pan with mouse
- **Legend**: Toggle cycle groups on/off

### Cycle Coloring

Beads are colored by their cycle length (normalized by number of time slices):
- Each color represents one unique cycle length
- Helps identify permutation cycles in boson systems
- Cycle statistics shown in output

### Configuration Summary

For each configuration, the notebook displays:
- Number of time slices (M)
- Number of particles (N)
- Total number of cycles
- Unique cycle lengths
- Cycle size distribution

## File Format

The visualization works with worldline files in this format:

```
# PIMCID: identifier
# START_CONFIG 000000
      m        n   wlIndex       x            y            z        prev_m   prev_n   next_m   next_n
      0        0        1       0.000E+00    0.000E+00    0.000E+00    2        0        1        0
      ...
# END_CONFIG
```

This format is compatible with:
- PIMC simulation output
- Minimization trajectory output from `find_local_minimum()`

## Troubleshooting

**No configurations loaded:**
- Check file path is correct
- Verify file contains `START_CONFIG` / `END_CONFIG` markers

**Empty plot:**
- Ensure box dimensions are set correctly
- Check if configurations are closed worldlines (open worldlines are filtered)

**Import errors:**
- Make sure you're running from the correct directory
- The notebook adds `../..` to path to import from `jax_landscape`
